# Tic Tac Toe Game in C

## 📌 Project Description

This is a simple 2-player Tic Tac Toe game developed in C.
It allows users to play the game in terminal and includes save and load functionality.

## ✨ Features

* Two-player gameplay
* Win detection system
* Save game to file
* Load previously saved game

## 🛠️ Technologies Used

* C Programming
* File Handling
* Arrays and Functions
* Structures

## ▶️ How to Run

1. Compile the code:
   gcc tictactoe.c -o game

2. Run the program:
   ./game
